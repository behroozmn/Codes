# document

* کل صفحه از بالا تا پایین با تمام اسکرول‌های آن

```jquery
$(document).ready(function (){ //Ready: -> وقتی صفحه کامل لود شد تابع را اجرا بنماید
    $("#ID").fadeOut(3000);    // المان را در ۳ثانیه حذف نماید
});
```
